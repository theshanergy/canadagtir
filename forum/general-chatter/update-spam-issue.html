Update on spam issue..

Well folks we're trying something new - all forum posts and comments are now run through a service called 'mollom', which checks their contents for spam type wording and presents a few deterents if necessary. This applies to the new user registration form as well.

I've never used this particular service before, so this is just a test. I do have the ability to override molloms spam testing all together on a per-account basis, so if people want to start making a list here of users we know to be 'real' I can start switching their accounts.

I hope this cuts down on some of the spam we've seen pouring in as of late!

 

Thanks,
