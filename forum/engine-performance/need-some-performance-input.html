I've got NIStune , if you're interested you could probably order from a US supplier rather than directly from OZ to save yourself some freight. 

My non-professional opinion regarding performance is that the amount of additional power (and reliability) you're looking for will determine the type of modifications. For massive performance gains, you might want to consider forged internals. 'Smaller' performance gains you can tinker with ecu, fuel supply etc.

I'd def start with the fuel pump.
